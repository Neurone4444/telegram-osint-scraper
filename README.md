# # Telegram OSINT Scraper


Offline tool for scraping public Telegram channels and producing OSINT reports:
triage, topics, spikes, dedup and incident cards.

## Origin & Attribution

This project is an independent evolution of the following open-source work:

- https://github.com/DarkWebInformer/telegram-scraper  
  (originally forked from https://github.com/unnohwn/telegram-scraper)

The codebase has been significantly extended and modified, including:
- offline AI analysis
- OSINT-focused triage and scoring
- anomaly detection and clustering
- reporting pipeline and OPSEC hardening
